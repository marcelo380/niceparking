### Sobre

:parking: NiceParking é um projeto que visa controlar as entradas e saida de veículos de um estacionamento.
### Funcionalidades 

→ Adicionar veículo na vaga.  
→ Remover veículo da vaga.  
→ Relatório de veículos por data.  
### Tecnologias

→ sqlite  
→ dart / flutter
